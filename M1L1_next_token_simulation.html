🔢 One token at a time

Each predicted token becomes context for the next prediction. This chain is called autoregressive generation — the model never sees the whole answer at once.

📊 Probability, not knowledge

The model calculates which token is statistically most likely — not what is objectively true. This is why confident-sounding AI answers can still be wrong.
